<p>Aspose.PSD für .NET ist eine erweiterte PSD- und AI-Dateiformat-Manipulations-API der Einstiegsklasse ohne Abhängigkeiten von Adobe Photoshop oder Adobe Illustrator. Die API ermöglicht das Erstellen und Bearbeiten der Photoshop-Dateien sowie die Möglichkeit, Ebeneneigenschaften zu aktualisieren, Wasserzeichen hinzuzufügen, Grafikoperationen durchzuführen oder ein Dateiformat in ein anderes zu konvertieren.</p>

<p>
 Die API unterstützt den Import als Layer und den Export in die folgenden Formate:
 <a href="https://docs.fileformat.com/image/png/">
  PNG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/jpeg/">
  JPEG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/j2k/">
  JPEG2000
 </a>
 ,
 <a href="https://docs.fileformat.com/image/gif/">
  GIF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/bmp/">
  BMP
 </a>
 ,
 <a href="https://docs.fileformat.com/image/tiff/">
  TIFF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/psd/">
  PSD
 </a>
 und
 <a href="https://docs.fileformat.com/image/psb/">
  PSB
 </a>
 zusammen mit dem Export nach Pdf mit auswählbarem Text. Aspose.PSD unterstützt teilweise Adobe Photoshop-Funktionen wie Anpassungsebenen, Rendern und Bearbeiten von Textebenen, Füllebenen, Ebeneneffekte, Bearbeiten von Raster- und Vektormasken für Ebenen.
</p>
