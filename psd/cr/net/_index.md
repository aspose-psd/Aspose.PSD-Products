<p>Aspose.PSD za .NET je napredni PSD i API za manipulaciju formatima datoteka ulazne razine bez ikakvih ovisnosti o Adobe Photoshopu ili Adobe Illustratoru. API omogućuje stvaranje i uređivanje datoteka Photoshopa, kao i mogućnost ažuriranja svojstava sloja, dodavanja vodenih žigova, izvođenja grafičkih operacija ili pretvaranja jednog formata datoteke u drugi.</p>

<p>
 API podržava uvoz kao sloj i izvoz u sljedeće formate:
 <a href="https://docs.fileformat.com/image/png/">
  PNG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/jpeg/">
  JPEG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/j2k/">
  JPEG2000
 </a>
 ,
 <a href="https://docs.fileformat.com/image/gif/">
  GIF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/bmp/">
  BMP
 </a>
 ,
 <a href="https://docs.fileformat.com/image/tiff/">
  TIFF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/psd/">
  PSD
 </a>
 i
 <a href="https://docs.fileformat.com/image/psb/">
  PSB
 </a>
 zajedno s izvozom u PDF s tekstom koji se može odabrati. Aspose.PSD djelomično podržava značajke Adobe Photoshopa kao što su slojevi za prilagodbu, renderiranje i uređivanje slojeva teksta, slojevi popunjavanja, efekti slojeva, uređivanje rasterskih i vektorskih maski za slojeve.
</p>
