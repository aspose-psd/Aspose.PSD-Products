Файли з розширенням .BMP представляють файли растрових зображень, які використовуються для зберігання растрових цифрових зображень. Ці зображення не залежать від графічного адаптера і також називаються форматом файлу незалежних від пристрою растрового зображення (DIB). Ця незалежність призначена для відкриття файлу на кількох платформах, таких як Microsoft Windows і Mac. Формат файлу BMP може зберігати дані у вигляді двовимірних цифрових зображень як у монохромному, так і в кольоровому форматі з різною глибиною кольору.
