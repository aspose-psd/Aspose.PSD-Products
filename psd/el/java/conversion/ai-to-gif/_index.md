Ένα κινούμενο GIF συνδυάζει πολλές εικόνες ή καρέ σε ένα ενιαίο αρχείο και τα εμφανίζει σε μια ακολουθία για να δημιουργήσει ένα κινούμενο κλιπ ή ένα σύντομο βίντεο.
