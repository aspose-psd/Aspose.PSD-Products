<p>Το Aspose.PSD για .NET είναι ένα προηγμένο PSD και αρχικού επιπέδου AI μορφές API χειρισμού αρχείων χωρίς εξαρτήσεις Adobe Photoshop ή Adobe Illustrator. Το API επιτρέπει τη δημιουργία και επεξεργασία των αρχείων Photoshop, καθώς και την παροχή της δυνατότητας να ενημερώσετε τις ιδιότητες του επιπέδου, να προσθέσετε υδατογραφήματα, να εκτελέσετε λειτουργίες γραφικών ή να μετατρέψετε μια μορφή αρχείου σε μια άλλη.</p>

<p>
 Το API υποστηρίζει την εισαγωγή ως επίπεδο και εξαγωγή στις ακόλουθες μορφές:
 <a href="https://docs.fileformat.com/image/png/">
  PNG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/jpeg/">
  JPEG
 </a>
 ,
 <a href="https://docs.fileformat.com/image/j2k/">
  JPEG2000
 </a>
 ,
 <a href="https://docs.fileformat.com/image/gif/">
  GIF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/bmp/">
  BMP
 </a>
 ,
 <a href="https://docs.fileformat.com/image/tiff/">
  TIFF
 </a>
 ,
 <a href="https://docs.fileformat.com/image/psd/">
  PSD
 </a>
 και
 <a href="https://docs.fileformat.com/image/psb/">
  PSB
 </a>
 μαζί με εξαγωγή σε Pdf με επιλέξιμο κείμενο. Το Aspose.PSD υποστηρίζει εν μέρει τέτοιες λειτουργίες του Adobe Photoshop, όπως επίπεδα ρύθμισης, απόδοση και επεξεργασία επιπέδου κειμένου, επίπεδα γεμίσματος, εφέ επιπέδων, επεξεργασία ράστερ και διανυσματικές μάσκες για επίπεδα.
</p>
