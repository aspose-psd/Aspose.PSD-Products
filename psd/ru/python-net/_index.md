<p>Aspose.PSD для Python через .NET — это уникальная библиотека Python PSD, предлагающая расширенные возможности обработки файлов PSD, PSD и AI. С помощью этого API вы можете легко создавать, загружать, обновлять, редактировать, конвертировать, сжимать изображения PSD и PSB. Aspose.PSD поддерживает самые популярные функции для файлов PSD и PSB, включая обновления текстовых слоев, смарт-объектов, слоев заливки, слоев-фигур, групповых слоев, корректирующих слоев. Aspose.PSD поддерживает режимы наложения, эффекты слоев, варп-преобразования, интеллектуальные фильтры, анимацию TimeLine, работу с векторными, растровыми и обтравочными масками, низкоуровневое изучение ресурсов PSD-файлов и многое другое</p>
