<p>
 Aspose.PSD для Java — это простой в использовании API управления форматами файлов Adobe Photoshop. Он может легко загружать и читать файлы PSD, PSB и AI, что позволяет разработчикам Java выполнять такие операции, как обновление свойств слоев, добавление водяных знаков, сжатие, поворот, масштабирование или рендеринг одного формата файла в другой без необходимости установки Adobe Photoshop. Форматы изображений, такие как JPEG, TIFF, GIF, PNG, BMP, JPEG2000, могут быть легко открыты и добавлены в виде слоя всего за несколько строк кода. Файлы PSD, PSB и AI можно экспортировать в указанный формат и дополнительно в формат PDF
</p>
