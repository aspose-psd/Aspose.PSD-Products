        {{< blocks/products/pf/agp/about-file-text fileFormat="PSD" readMoreLink="https://docs.fileformat.com/image/psd/" >}}

PSD, Photoshop Document, reprezentas la indiĝenan dosierformaton de Adobe Photoshop uzata por grafika dezajnado kaj disvolviĝo. PSD-dosieroj povas inkluzivi bildavolojn, alĝustigajn tavolojn, tavolmaskojn, komentadojn, dosierojn, ŝlosilvortojn kaj aliajn Photoshop-specifajn elementojn. Photoshop-dosieroj havas defaŭltan etendon kiel .PSD kaj havas maksimuman altecon kaj larĝon de 30,000 pikseloj, kaj longolimon de du gigabajtoj.

        {{< blocks/products/pf/agp/about-file-text fileFormat="PNG" readMoreLink="https://docs.fileformat.com/image/png/" >}}

PNG, Portable Network Graphics, rilatas al speco de rastruma bilda dosierformato, kiu uzas senperdan kunpremadon. Ĉi tiu dosierformato estis kreita kiel anstataŭaĵo de Graphics Interchange Format (GIF) kaj ne havas kopirajtajn limojn. Tamen, PNG-dosierformato ne subtenas kuraĝigojn. PNG-dosierformato subtenas senperdan bildkunpremadon, kiu popularigas ĝin inter siaj uzantoj. Kun la paso de tempo, PNG evoluis kiel unu el la plej uzataj bildaj dosierformatoj. Preskaŭ ĉiuj Operaciumoj havas subtenon por malfermi PNG-dosierojn. Ekzemple, Microsoft Windows-spektilo havas la kapablon malfermi PNG-dosierojn ĉar la OS havas defaŭlte la subtenon disponeblan kiel parto de instalado.
