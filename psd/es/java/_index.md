<p>
 Aspose.PSD para Java es una API de manipulación de formatos de archivos de Adobe Photoshop fácil de usar. Puede cargar y leer fácilmente archivos PSD, PSB e AI, lo que permite a los desarrolladores de Java realizar operaciones como actualizar las propiedades de las capas, añadir marcas de agua, comprimir, girar, escalar o renderizar un formato de archivo a otro sin necesidad de instalar Adobe Photoshop. Formatos de imagen como JPEG, TIFF, GIF, PNG, BMP, JPEG2000 y se pueden abrir y añadir fácilmente como una capa con solo unas pocas líneas de código. Los archivos PSD, PSB y AI se pueden exportar al formato de la lista y, además, al formato PDF
</p>
