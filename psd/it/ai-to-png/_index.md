PNG, Portable Network Graphics, si riferisce a un tipo di formato di file di immagine raster che utilizza una compressione senza perdite. Questo formato di file è stato creato in sostituzione del Graphics Interchange Format (GIF) e non presenta limitazioni di copyright. Tuttavia, il formato di file PNG non supporta le animazioni. Il formato di file PNG supporta la compressione delle immagini senza perdite che lo rende popolare tra i suoi utenti. Con il passare del tempo, il PNG si è evoluto come uno dei formati di file di immagine più utilizzati. Quasi tutti i sistemi operativi supportano l'apertura di file PNG.
