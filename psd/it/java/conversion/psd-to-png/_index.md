        {{< blocks/products/pf/agp/about-file-text fileFormat="PSD" readMoreLink="https://docs.fileformat.com/image/psd/" >}}

PSD, Photoshop Document, rappresenta il formato di file nativo di Adobe Photoshop utilizzato per la progettazione e lo sviluppo di grafica. I file PSD possono includere livelli di immagine, livelli di regolazione, maschere di livello, annotazioni, informazioni sui file, parole chiave e altri elementi specifici di Photoshop. I file di Photoshop hanno l'estensione predefinita come .PSD e hanno un'altezza e una larghezza massime di 30.000 pixel e un limite di lunghezza di due gigabyte.

        {{< blocks/products/pf/agp/about-file-text fileFormat="PNG" readMoreLink="https://docs.fileformat.com/image/png/" >}}

PNG, Portable Network Graphics, si riferisce a un tipo di formato di file di immagine raster che utilizza una compressione senza perdite. Questo formato di file è stato creato in sostituzione del Graphics Interchange Format (GIF) e non presenta limitazioni di copyright. Tuttavia, il formato di file PNG non supporta le animazioni. Il formato di file PNG supporta la compressione delle immagini senza perdite che lo rende popolare tra i suoi utenti. Con il passare del tempo, il PNG si è evoluto come uno dei formati di file di immagine più utilizzati. Quasi tutti i sistemi operativi supportano l'apertura di file PNG. Ad esempio, il visualizzatore Microsoft Windows ha la capacità di aprire file PNG poiché il sistema operativo dispone, per impostazione predefinita, del supporto disponibile come parte dell'installazione.
