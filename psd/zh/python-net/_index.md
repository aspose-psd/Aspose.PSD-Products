<p>Aspose.PSD for Python via .NET 是一个独特的 Python PSD 库，提供高级 PSD、PSD 和 AI 文件处理功能。您可以使用此 API 轻松创建、加载、更新、编辑、转换、压缩 PSD 和 PSB 图像。Aspose.PSD 支持 PSD 和 PSB 文件的最常用功能，包括文本图层、智能对象、填充图层、形状图层、群组图层、调整图层的更新。Aspose.PSD 支持混合模式、图层效果、变形变换、智能滤镜、动画时间轴、使用矢量、栅格和剪贴蒙版、低级 PSD 文件资源探索等</p>

<ul>
