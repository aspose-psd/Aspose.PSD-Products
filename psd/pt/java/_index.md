<p>
 O Aspose.PSD para Java é uma API de manipulação de formato de arquivo do Adobe Photoshop fácil de usar. Ele pode carregar e ler facilmente arquivos PSD, PSB e AI, possibilitando que os desenvolvedores de Java realizem operações como atualizar as propriedades da camada, adicionar marcas d'água, compressão, rotação, dimensionamento ou renderização de um formato de arquivo para outro sem precisar instalar o Adobe Photoshop. Formatos de imagem como JPEG, TIFF, GIF, PNG, BMP, JPEG2000 podem ser facilmente abertos e adicionados como uma camada com apenas algumas linhas de código. Arquivos PSD, PSB e AI podem ser exportados para o formato listado e, adicionalmente, para o formato PDF
</p>
